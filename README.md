🎮 Number Guessing Game (C)

🚀 A fun and interactive console-based game built using C, where the player tries to guess a randomly generated number with smart hints and multiple difficulty levels.
---
✨ Features

- 🎯 Random number generation using "rand()" and "srand()"
- 🎚 Multiple difficulty levels:
  - 🟢 Easy (1–50)
  - 🟡 Medium (1–100)
  - 🔴 Hard (1–500)
- 📊 Attempt counter
- 💡 Smart hints:
  - Too High 📈
  - Too Low 📉
- 🏆 Winning message with total attempts

---

🧠 Concepts Used

- C Programming Basics
- Loops ("do-while")
- Conditional Statements ("if-else")
- Functions ("rand", "srand", "time")
- User Input Handling

---

🎮 How to Play

1. Run the program
2. Choose your difficulty level
3. Enter your guess within the given range
4. Follow hints to reach the correct number
5. Win the game 🎉

---

▶️ How to Run

gcc main.c -o game
./game
